# To-Do-App-in-React with Redux & Local Storage

### Features :

<li>Add Todo</li>
<li>Edit & update Todo</li>
<li> Delete Todo</li>

<li>Search(filter) Todo</li>
<li>Check individual Todo when Completed</li>
<li>Remember Todo</li>
